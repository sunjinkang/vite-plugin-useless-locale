# vite-plugin-useless-locale

用于展示项目的语言包中未使用的数据

## 引用

```js
import uselessLocale from 'vite-plugin-useless-locale';
```

### 属性

> 可选参数

- `localePath` - 语言包文件路径

| 属性名称    | 属性类型                                                 | 属性值 | 是否必选项 |
| ----------- | -------------------------------------------------------- | ------ | ---------- |
| localePath          | string                                                   | -      | +          |
